# Who are we?

พวกเราเป็น First Voter ที่ผลิตสื่อขึ้นมาเพื่อจำลองขั้นตอนการเลือกตั้งตามกติกาใหม่และรวบรวมข้อมูลที่จำเป็นต่าง ๆ เกี่ยวกับการเลือกตั้งสำหรับทุกคน !
